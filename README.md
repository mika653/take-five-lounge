# Take Five — Landing Page Mockup

A concept mockup of a **dramatic, sexy, premium** landing page for **Take Five**, the intimate live-jazz cocktail lounge inside Avalon at H Queen's, 80 Queen's Road Central, Hong Kong (an Elite Concepts venue).

The most important feature is the **table booking flow** — a 4-step reservation drawer (When & Who → Seating → Details → Confirmation), reachable from a persistent hero CTA and a sticky mobile bar.

## What's inside
- `index.html` — the entire page, self-contained (no build step, no dependencies beyond Google Fonts).

## Run locally
Just open `index.html` in a browser, or serve it:

```bash
npx serve .
```

## Content
Real venue details: five signature cocktails (HK$110), Skip Moy's music programme, Sandeep Kumar's bar, Jarrod Verbiak's kitchen, address and hours.

> Concept mockup for presentation only — the booking flow is a front-end demo and does not make live reservations.
